===== 1. create new component ====

ng g c header --dry-run
  OR
ng g c header
  OR
ng generate component header
// it will create inside `src/app` folder


/*** you can also create in a nested folder ***/

ng g c component/header
// it will create inside `src/app/components` folder


ng g c home 
// creates the HomeComponent
